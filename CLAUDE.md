# CLAUDE.md

## Project Overview

- **프로젝트명:** basemsg
- **비전:** 누구나 쉽게 사용할 수 있는 실시간 메신저 플랫폼
- **비즈니스 밸류:** 모바일 + 웹 크로스플랫폼 메신저로, 개인/소규모 팀 커뮤니케이션 지원
- **목적:** 실시간 채팅, 파일 공유, 친구 관리 기능을 제공하는 메신저 앱 서비스
- **서비스 대상:** 일반 사용자 (모바일 Android/iOS + 웹)
- **개발 범위:** 모바일 앱, 웹 프론트엔드, NestJS 백엔드, 실시간 통신, 인증, 배포 인프라
- **GitHub:** https://github.com/muje2002/basemsg-firebase-claude

## 프로젝트 문서 체계

아래 MD 파일들을 **모든 작업 시 항상 참조**하고, 변경 사항이 있으면 해당 파일을 **즉시 업데이트**할 것.

| 파일 | 역할 | git 반영 |
|------|------|----------|
| `CLAUDE.md` | 프로젝트 개요 + 문서 체계 안내 (이 파일) | O |
| `config_deploy.md` | 개발/배포 환경 설정값, 계정 정보, 토큰 등 | **X (gitignore)** |
| `rules.md` | 코드 컨벤션, 테스트 정책, CI/CD 규칙 등 글로벌 규칙 | O |
| `memory.md` | 개발 히스토리, 현재 상태, 다음 작업 목표 (세션 끊김 대비) | O |
| `SW.md` | SW 아키텍처, 기술 스택, API 스펙, DB 스키마, UX/UI 설계 | O |

### 작업 규칙
1. 코드 작업 전 `rules.md` 확인 (컨벤션, 테스트 정책)
2. 구현 시 `SW.md` 참조 (아키텍처, API 스펙, DB 스키마)
3. 설정/배포 관련 작업 시 `config_deploy.md` 참조
4. 작업 시작/완료 시 `memory.md` 업데이트
5. SW 구조나 스펙 변경 시 `SW.md` 업데이트
